# Run report — `ace-downscaling-distillation-fdistill-with-val-prate-spectral-gamma1`

_Hypothesis: same as the `gamma0p5` arm but a stronger high-k tilt. `band_gamma=1`
applies `band_weight_k ∝ (k/k_max)^1`, normalized to mean 1, giving ≈ lo 0.33× /
mid ~1.0× / hi 1.7× — a linear ramp that emphasizes the tail while still keeping a
(reduced) low-band constraint. Expectation: larger `spec_mae_hi_PRATEsfc` improvement
than `gamma0p5`, at the cost of some lo. Paired with `gamma0p5` this yields a
`band_gamma ∈ {0, 0.5, 1}` response curve (0 = `i26sidsm`; the low-suppression endpoint
~2 is already covered by `xgcaf2rt`), which distinguishes "emphasize hi" (good) from
"starve lo" (the neutral `xgcaf2rt` failure mode)._
