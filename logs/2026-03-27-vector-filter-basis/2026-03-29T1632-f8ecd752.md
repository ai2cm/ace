> ## 2. Key Technical Concepts
> 
> - **Full dry-adiabatic thermodynamic equation**: `∂T/∂t = -V·∇T + ω·(κT/p − ∂T/∂p)`, derived from dθ/dt=0 (potential temperature conservation)
> - **Continuity in pressure coords**: `∂ω/∂p = -∇·V`, integrated upward from surface (ω=0)
> - **Divergence operator**: `W_vs[0,0,:,1] = 1/R_EARTH` (confirmed from ShallowWaterStepper convention; index 1 = divergence, index 0 = vorticity)
> - **∇^(2n) diffusion**: `_laplacian = _divergence(_gradient(s))`; sign `(-1)^(n+1)`; applied component-wise to V, T, q
> - **JW2006 test case**: Balanced mid-latitude jet; T from JW eq. 6 directly (NOT hydrostatic inversion); u from JW eq. 2; 1K temperature perturbation seeds instability
> - **Super-adiabatic instability**: The JW hydrostatic-inversion initialization gives `∂T/∂p ≈ 0.0016 K/Pa > κT/p ≈ 0.0010 K/Pa` at equatorial surface → convectively unstable; large-scale blow-up, not curable by ∇² diffusion at ν < 5e7
> - **Sigma coordinates (σ = p/p_s)**: p_s is prognostic; PGF gains `-RT∇(ln p_s)` correction term; σ̇ replaces ω; hydrostatic formula identical to isobaric (p_s cancels in ln ratio)
> - **σ̇ diagnostic**: `σ̇_{k+1/2} = σ̇_{k-1/2} - [div_k + V_k·∇(ln p_s)] Δσ_k`
> - **Surface pressure tendency**: `∂p_s/∂t = -p_s ∑_k [div_k + V_k·∇(ln p_s)] Δσ_k`

## Hybrid Sigma-Pressure Coordinate Formulation

**Pressure at each level:**
```
p_k(λ,φ,t) = a_k + b_k · p_s(λ,φ,t)
```
`a_k` (Pa) and `b_k` (dimensionless) are prescribed constants. At the surface interface `b=1, a=0`; at the top interface `b=0, a=p_top`. The pure-sigma case is `a=0, b=σ_k`.

**Layer thicknesses at interfaces:**
```
Δa_k = a_{k+1/2} - a_{k-1/2}    (constant)
Δb_k = b_{k+1/2} - b_{k-1/2}    (constant)
Δp_k = Δa_k + Δb_k · p_s         (B,K,H,W — varies in space)
```

---

### 1. Surface pressure tendency

```
∂p_s/∂t = −∑_k ( Δp_k div_k  +  Δb_k V_k·∇p_s )
```

Reduces to the sigma formula when `Δa=0, Δb=Δσ, Δp=Δσ·p_s`.

---

### 2. η̇ recurrence (vertical velocity diagnostic)

Define `C_{k+1/2} = (∂p/∂η) η̇` at interfaces. Starting from `C_{−1/2} = 0` (top BC):

```
C_{k+1/2} = C_{k-1/2}  −  Δb_k ∂p_s/∂t  −  Δp_k div_k  −  Δb_k V_k·∇p_s
```

Mid-level: `C_k = ½(C_{k-1/2} + C_{k+1/2})`.

Note: the existing `SigmaCoordinateStepper._sigma_dot` is missing the `Δb_k ∂p_s/∂t` term (equivalent to `Δσ_k ∂p_s/∂t`). This is a bug — small but worth fixing in the hybrid version from the start.

---

### 3. Pressure gradient force

```
PGF_k = −∇φ_k  −  (R T_k b_k / p_k) ∇p_s
```

In sigma: `b_k/p_k = σ_k/(σ_k p_s) = 1/p_s`, so the correction becomes `−RT ∇(ln p_s)` ✓.

The geopotential `φ_k` is now a full spatial field because `ln(p_{k-1}/p_k) = ln((a_{k-1}+b_{k-1}p_s)/(a_k+b_k p_s))` varies with `p_s`. Apply the DISCO gradient operator to the resulting (B,K,H,W) field — no structural change needed.

---

### 4. Temperature tendency

The pressure velocity `ω_k = dp/dt`:
```
ω_k = b_k(∂p_s/∂t + V_k·∇p_s) + C_k
```

Temperature tendency:
```
∂T/∂t = −V·∇T  +  (κT/p)ω  −  C_k ∂T/∂p
```

Here `∂T/∂p` uses central finite differences in pressure-space:
```
∂T/∂p|_k = (T_{k+1} − T_{k-1}) / (p_{k+1} − p_{k-1})
```
where `p_{k±1} = a_{k±1} + b_{k±1} p_s` is a spatially-varying field. Similarly for `∂V/∂p` in the momentum vertical advection and `∂q/∂p` in humidity.

In sigma this reduces correctly: `b/p = 1/p_s`, `ω = (∂p_s/∂t + V·∇p_s)/p_s · p + C`, and everything matches.

---

### 5. Hydrostatics

```
φ_0 = φ_surface
φ_k = φ_{k-1}  +  R T_{k-1}  ln(p_{k-1}/p_k)
```

Unlike sigma, `ln(p_{k-1}/p_k)` is no longer a constant — it's a (B,H,W) field computed each step from `p_k = a_k + b_k p_s`.
